# 🌸 Iris Flower Classification

A professional Machine Learning project using the Iris dataset to classify flowers into different species.

---

## 🚀 Features

- Data preprocessing
- Exploratory Data Analysis (EDA)
- Visualization
- Feature scaling
- Model training
- Performance evaluation
- Confusion matrix
- Model saving/loading

---

## 🛠 Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

---

## 📂 Dataset

The dataset used is the famous Iris dataset available in Scikit-learn.

Classes:
- Setosa
- Versicolor
- Virginica

---

## 📊 Model Used

Random Forest Classifier

---

## 📈 Accuracy

Achieved approximately 96%–100% accuracy depending on train-test split.
