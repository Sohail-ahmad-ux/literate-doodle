# 📌 Project Overview

The **Iris Flower Classification System** is a beginner-friendly yet professional-level Machine Learning project built using Python and Scikit-learn.

This project demonstrates:

✅ Data Preprocessing  
✅ Exploratory Data Analysis (EDA)  
✅ Feature Scaling  
✅ Data Visualization  
✅ Model Training  
✅ Model Evaluation  
✅ Confusion Matrix Visualization  
✅ Model Saving & Loading

The dataset used is the famous **Iris Dataset** available in Scikit-learn.
